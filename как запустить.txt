1) Создайте новый проект с помощью Create React App:

npx create-react-app news-feed

2) Перейдите в созданную папку проекта:

cd news-feed

3) Установите зависимости для React Router:

npm install react-router-dom

4) Замените папку news-feed по данному пути на мою (C:\Users\admin(ваше имя пользователя)).

Запустите приложение:

npm start

Откройте браузер и перейдите по адресу http://localhost:3000 (если не открылась сама)
